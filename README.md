# EcoCoil Mobile App

Мобильное приложение для системы очистки воздуха нового поколения на солнечной энергии. Приложение предназначено для мониторинга качества воздуха (PM-2.5) в реальном времени, а также отслеживания состояния батареи и солнечной панели.

## 📱 Как запустить проект

Данный репозиторий содержит **ядро** Flutter-приложения (исходный код в папке `lib` и зависимости `pubspec.yaml`). Чтобы запустить его у себя локально на Android, iOS или Web, выполните следующие шаги:

### 1. Предварительные требования
Убедитесь, что у вас установлен [Flutter SDK](https://docs.flutter.dev/get-started/install).

### 2. Клонирование репозитория
Скачайте проект к себе на компьютер:
```bash
git clone https://github.com/Abdihamitov/eccoil_app.git
cd eccoil_app
```

### 3. Генерация платформенных файлов (Важно!)
Поскольку в репозитории нет платформенных папок (`android`, `ios`, `web` и т.д.), вам нужно их сгенерировать. Выполните в корне проекта:
```bash
flutter create .
```
*Эта команда безопасно создаст недостающие файлы для запуска, не затрагивая написанный UI-код в папке `lib/`.*

### 4. Установка зависимостей
Скачайте нужные пакеты (графики `fl_chart`, шрифты `google_fonts`):
```bash
flutter pub get
```

### 5. Запуск
Подключите телефон, запустите эмулятор (Android/iOS) или откройте Chrome, и выполните:
```bash
flutter run
```
